Snake River Smart Homes

Site Plan

Site Name

Snake River Smart Homes

This name was chosen because it sounds professional, local, and memorable. It connects the business to the Snake River region while clearly communicating that the company focuses on smart home services.

Purpose

Snake River Smart Homes is a business website designed to help customers learn about smart home technology and professional installation services. The site allows visitors to explore services, compare smart home packages, and use an interactive tool to find the best setup for their home and priorities.

Audience

The target audience includes homeowners, renters, families, and individuals who want to improve convenience, security, and efficiency through smart home technology. The site is especially intended for people who may be interested in smart home systems but want simple explanations, package options, and guided recommendations before making a decision.

Scenarios

  • What smart home package would work best for my home type and main priority?
  • What services does Snake River Smart Homes offer for security, lighting, and automation?
  • How can I request a quote for a smart home setup that fits my budget and needs?

Dynamic Elements

The site includes an interactive Smart Home Package Finder built with JavaScript. Users choose a home type and top priority, and the site responds by recommending a package with an estimated starting price and description.

The site also includes a contact form with JavaScript-based interaction that responds to user input and displays a confirmation message after submission.

The JavaScript will include the following required elements:

  • More than one function to organize the code clearly.
  • DOM interaction by selecting elements, updating text and content, and responding to button clicks and form submissions.
  • Conditional branching using if/else logic to determine the best package recommendation based on the user’s choices.
  • Objects and arrays to store package information and form-related data.
  • Array methods such as filter() and find() to process package recommendation logic.

Logo

The site uses a simple house-style logo mark paired with the business name Snake River Smart Homes. The logo is designed to feel clean, modern, and professional while supporting the smart home theme.

Logo Style

Simple house icon or home symbol paired with modern text styling.

Background Color

The site uses a dark charcoal and deep gray base to create a modern, premium smart-home feel. A green accent color is used for buttons, highlights, and important interactive elements. A soft light background keeps content sections clean, readable, and professional.

Typography

Poppins will be used for headings because it looks modern and strong.

Roboto will be used for body text because it is clean and easy to read.

Content

Home Page

Hero Heading: Smarter security, comfort, and control for your home

The homepage introduces the business, explains the value of smart home technology, previews services and packages, and encourages users to explore the rest of the site.

Services Page

This page explains the company’s main service categories, including smart lighting, home security, cameras and doorbells, automation, and voice assistant integration. Each section includes practical descriptions and use cases.

Packages Page

This page compares the Starter, Security, and Premium packages. It explains what each package includes, who it is best for, and how customers can use the interactive package finder to choose an option.

Contact Page

The contact page includes a quote request form where users can enter their name, email, home type, area of interest, and project details.

Interactive Tool

Users can choose a home type and a top priority to receive a recommended package with pricing and description updates on the page.
